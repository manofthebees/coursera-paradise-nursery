Paradise Nursery is a React-based shopping application for houseplants.
Users can browse plants by category, add them to a cart, update quantities,
and view total costs before checkout.

## Features
- Landing page with background image
- Product listing with categories
- Redux-powered shopping cart
- Dynamic cart quantity & pricing

## Tech Stack
- React
- Redux Toolkit
- React Router
- GitHub Pages
